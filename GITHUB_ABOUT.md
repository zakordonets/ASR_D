# GitHub About

## Short Description

CLI for transcription, diarization, subtitle export, and optional LLM normalization using GigaAM, pyannote, DeepSeek, and OpenRouter.

## Extended Description

Python CLI utility for audio and video transcription with speaker diarization, optional transcript normalization via LLM, and export to `txt`, `json`, `srt`, and `vtt`. Supports single files, combined files, and batch folder processing with failure isolation.
